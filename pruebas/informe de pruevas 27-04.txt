prueva de la actualizacion de un perfil

- nota para el frontend para poder actualiar un perfil primero se debe obtener un token para que se verifique 
el usuario antes que se pueda realizar la accion de modificarlo.la llava del token es 
'Authorization Bearer token_recibido' 

- la primer prueba fue un exito el perfil se actualizo demanera correcta , en este primer intento intenter modificar 
el nombre,estilo de vida ,telefono y presupusto, anque pensano lo mejor el presuypuesto deberia ser un valorque se 
pone en cada vez que se quiere armar un paquete 

- bien las prebas para modificar el perfil solo con uno o mas variables fue un exito lo permite sin ningun 
problema no hay grandes cosas por poner o destacar todo funciona correctamente

- pero como nota se deberia de poner una restriccion que impida el uso de nombres inapropiados
